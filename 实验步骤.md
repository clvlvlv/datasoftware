# 数据备份软件开发综合实验 —— 实验步骤文档

## 一、项目概述

### 1.1 实验目标
设计并实现一款数据备份软件，支持目录树文件的备份与还原，并可扩展文件类型支持、元数据支持、自定义备份、打包解包、压缩解压、加密解密、图形界面等功能。

### 1.2 开发环境
- **操作系统：** Windows 10/11
- **语言：** C++17
- **构建工具：** CMake + Visual Studio 2019
- **GUI 框架：** Qt 5.15
- **版本控制：** Git + GitHub
- **第三方库：** Windows BCrypt API（加密）

#### 步骤 2.9：实现压缩算法
**文件：** `Compressor.h`、`Compressor.cpp`

**算法 1：RLE（Run-Length Encoding）**
- 原理：用 `[count, byte]` 替代连续重复字节
- 格式：DSRL magic + version + chunks
- 适用：含大量重复数据的文件

**算法 2：LZ77（Sliding Window）**
- 原理：用 `[offset, length]` 引用已出现的重复片段
- 窗口：4KB，预读：64 字节
- 格式：DSLZ magic + version + tokens

**算法 3：Huffman Coding**
- 原理：高频字符用短编码，低频用长编码
- 格式：DSHF magic + version + 编码表 + 位流
- 适用：通用数据（通常效果最好）

**测试：**
- RLE 往返 ✓
- LZ77 往返 ✓
- Huffman 往返 ✓
- 256 字节值全覆盖 ✓
- 大文件测试 ✓

4. 格式：DSENC magic + version + salt + IV + 加密数据 + HMAC

**测试：**
